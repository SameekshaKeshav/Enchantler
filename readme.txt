The files are to be run on the xampp cross-platform web server
which can be downloaded from https://www.apachefriends.org/download.html
The deployed application is copied into htdocs folder within the installed folder of xampp application
Start the necessary services like apache in the xampp UI
Deployed app can be accessed through http://localhost/app/
Application is better accessed via browser in incognito mode

Necessary python libraries for smooth running of the application:
	numpy: pip install numpy
	pandas: pip install pandas
	matplotlib: pip install matplotlib
	seaborn: pip install seaborn
